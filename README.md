# Consulta de CEP - API ViaCEP

Trabalho da disciplina SDM.

O programa consome a API ViaCEP e salva os dados retornados nos formatos JSON e XML.

## Tecnologias

* C#
* API ViaCEP

## Arquivos

* Endereco.cs → classe do endereço
* cep.json → retorno em JSON
* cep.xml → retorno em XML
* lista-01.pdf → respostas da atividade
